Version History

Current release · 1.2.4

June 8, 2026 Current

Versions 1.2.0–1.2.4

  • Help > About Setcraft now opens the standard About panel (it was an AppKit call made off the main thread, so it silently did nothing).
  • Removed the macOS window tab bar. Setcraft is a single-window app, so the default tabbing (the View > Show Tab Bar item and the empty tab strip) only added confusion.
  • Tutorial copy fixes: the overview's closing card now explains how to reach the demo band (via the Artist Switcher) rather than implying you are already in it; the songs "Type" step no longer claims covers print the original artist (they do not); the Backup tutorial highlights the Backup tab.
  • Lyric and chart books now print, not just PDF export. The desktop Print path honors the lyric/chart book checkboxes instead of dropping them.
  • Photo Import no longer shows "Setlist created" and "Review Matches" before you import anything (a desktop-only gate bug).
  • Backstage: the Account & License tab shows license status and the number of activated additional artist profiles; the Artists tab is wider so profile rows with long band names no longer wrap.
  • The Shows list now has a PRO Report link on each show with a setlist.
  • The dashboard rename (pencil) icon follows the theme color.
  • Tutorials: corrected the set cap (four, not unlimited), the default builder view (two sets, not one), and setlist saving (manual Save, not automatic); fixed the song filter list and the print-dialog wording. Added Lyrics, Charts, Shows filtering, and a Save step, a new Backup & Restore tutorial, and a data safety stop on the overview. Removed the redundant Run Setup Wizard entry.
  • Navigation and page labels are now consistently title-cased ("Build Setlist", "Photo Import") so they read correctly on themes whose fonts are not all-caps.
  • Automatic updates via Sparkle: the app checks the signed appcast feed (getsetcraft.com/appcast.xml) and can download, install, and relaunch a new version itself. Manual check from Help, Check for Updates.
  • Also carries the interim lightweight update check, which only runs on builds that do not have Sparkle, so there is never a double prompt.

June 7, 2026

Version 1.1.0

  • Local Backup: automatic on-device disaster-recovery snapshots, taken on launch and before any destructive action, with full rollback from the Backstage Backup tab.
  • Backstage reorganized into Preferences, Integrations, and Account & License.
  • Paid printouts no longer carry the free-version watermark.
  • A real upgrade prompt at the free song limit, replacing the plain error.
  • PRO Report reachable directly from the show page.
  • Accessibility: visible keyboard focus, screen-reader labels on icon buttons.
  • Broad copy and layout polish from the product audit.

June 6, 2026

Version 1.0.10

  • Final desktop setlist-drag fix (clone and slip behavior in the WKWebView).
  • License reclaim: "deactivate all other computers".
  • Free-tier nudge when over the 15-song limit.

June 2, 2026

Version 1.0.0

  • First signed and notarized release build, published to the free download channel and wired to Lemon Squeezy. Not distributed to anyone.

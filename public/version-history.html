Version History

Current release · 1.2.29

June 22, 2026 Current

Versions 1.2.27–1.2.29

  • Assorted visual polish and behind-the-scenes refinements.
  • The Lyric Book and Chart Book buttons now have their own clear descriptions.
  • Behind-the-scenes polish.

June 21, 2026

Versions 1.2.25–1.2.26

  • More of your personalization now carries over across app updates and backups.
  • Tuned one theme's background artwork for better readability.
  • Behind-the-scenes polish.
  • Backstage tabs and the Photo Import intro now sit on a solid panel, so they stay readable on every theme.

June 20, 2026

Versions 1.2.21–1.2.24

  • Several themes now have richer background artwork.
  • Behind-the-scenes polish.
  • Backstage Preferences now use the full window width, so the layout has more room to breathe.
  • The downloaded disk image now opens to a tidier install window, with the app and an Applications shortcut laid out for drag-to-install.
  • AirDrop: send a setlist, tech cue sheet, PRO report, lyric or chart book, or a single song's lyrics or chart straight to a nearby Mac, iPhone, or iPad — right from the Print options, without saving a file first.
  • The PRO Report can now be saved as a PDF (or AirDropped), not just printed.

June 19, 2026

Versions 1.2.17–1.2.20

  • Fixed an occasional macOS alert "beep" that could sound when typing while no text field was focused.
  • Your saved look and preferences now persist more reliably across app updates, backups, and a move to a new computer.
  • Assorted visual polish and behind-the-scenes refinements.
  • The small "?" help markers are gone. Tooltips now appear when you hover over a label, and a new switch in Backstage (Preferences) turns help tooltips on or off, so the screens stay uncluttered.
  • Printed setlists and exported PDFs now use one consistent font, regardless of the theme you have on in the app. No more setlists coming out in a pixel or script font just because of your current look.
  • Backstage: the theme picker tiles are more compact, so the list scrolls less.
  • The song editor (and other pop-up dialogs) open scrolled to the top, so the first field is in view instead of halfway down the page.
  • Added a built-in Help Center. Open it from the Help menu (Setcraft Help) and it comes up in its own window with a searchable set of guides: getting started, songs and lyrics, setlists and shows, tech cue sheets and printing, artist profiles, backup, themes, and a troubleshooting page. It works offline, so the answers are there even when the venue Wi-Fi is not.
  • Fixed a case where adding an artist profile showed the upgrade prompt even though you had a paid seat free. The demo band was being counted against your licensed limit; it no longer is, matching how the rest of the app counts.

June 18, 2026

Versions 1.2.14–1.2.16

  • Fixed license activation from the first-launch welcome screen. Entering a key and email on the onboarding "Activate" step failed with "Could not reach the activation server," even though the same key worked once you skipped to the free version and activated from Backstage. The onboarding guard was redirecting the activation request to the wizard page before it reached the license routes, so the response came back as HTML and the screen misread it as a connection failure. The /api/activate and /api/license/reclaim routes are now allowed to run during onboarding.
  • The onboarding wizard's "Pick a look" step now keeps its Back and Next buttons in view. The theme tiles were large enough that the full set pushed Next off the bottom of the window, which made it look like the buttons in the preview behind the picker were how you advanced. The tiles are now compact so everything fits, and the preview behind the picker no longer shows its (non-clickable) nav links, so there is nothing back there that looks like a way forward.
  • The top navigation links now show on the dashboard too, instead of disappearing there. Having them vanish on the landing page was more confusing than helpful.
  • The "Backstage" navigation item now carries a gear icon, so its settings purpose reads at a glance even if the name's backstage metaphor does not land (for example for users whose first language is not English).
  • Backstage Artists tab: the Primary and Active badges next to a profile name now sit in a tidy row, evenly spaced from the name and aligned at the same height.
  • The Bandsintown sync window now links to a short "How to connect Bandsintown" guide for getting your API key, and the free-version sync and Photo Import prompts open straight to the Account & License tab so a license key is one stop away. Links to outside pages now open in your browser instead of inside the app.
  • The demo band's name no longer shows a rename (pencil) button on the dashboard. The demo is a read-only sample, so its name was never editable; the button is gone to match.
  • Account & License: you can now enter an additional-artist-profile add-on key right on the Account & License tab. It used to live only inside the Manage Artists dialog, which is not where most people went looking for it.
  • When you have a paid artist seat you have not used yet, Backstage now points it out: a note on the Account & License tab and a small count badge on the Artists tab, so an activated-but-uncreated profile does not sit there unnoticed. The badge reads clearly on every theme.

June 16, 2026

Version 1.2.13

  • Free version: only your primary artist profile and the demo band stay available. Extra profiles are greyed in the switcher and need an active license. Choose which profile is your primary in Backstage, even on the free version.
  • Deactivating a license now warns you when you have extra profiles, and lets you set your primary first so you keep the right one.
  • Free version: a setlist is capped at 15 songs (your song library is unchanged; building longer setlists is part of the paid version). The demo band is exempt, so you can try full-length setlists there before you buy.
  • Free version: Bandsintown sync shows as locked.
  • The demo band is now a fuller, read-only sample (45 songs, many with lyrics and a sample number chart, plus a couple of saved reminder notes) and a complete sandbox: build multi-set setlists and tech cue sheets with it on the free version, and its exports print without the free-version watermark so you see the full output. Hide it (and bring it back) from Backstage instead of deleting it.
  • Free-version limits no longer show as persistent banners. The app stays out of your way and explains a limit only at the moment you reach one.

June 15, 2026

Version 1.2.12

  • The "Saved defaults" help tooltip in the builder's Between-Song Reminders panel no longer runs off the left edge of the window.
  • Removed a duplicate "View" menu from the menu bar. macOS already provides one with Enter Full Screen, so Reload now lives under the File menu.

June 14, 2026

Versions 1.2.10–1.2.11

  • The Tech Cue Sheet's "Back to Setlist" link now returns to the setlist builder, on the set you were working on, instead of the read-only show view.
  • "Edit Setlist" on the show view now opens that setlist in the builder ready to edit. It used to open an empty builder.
  • In-app email links (the beta feedback button, the Backstage feedback and add-on-code links, and the Licenses page contact) now open your mail client. They did nothing in the desktop app before.
  • In the setlist builder, the Between-Song Reminders panel moved from beneath the sets into the song library column, so saving reminders no longer shrinks the room for the setlist above it. The reminders and metaset panels are now collapsible, and a saved reminder can be added to the current set with a single click as well as by dragging it in.
  • The builder's saved-setlist bar gained a Tech Cue Sheet button, next to Print / PDF.

June 13, 2026

Version 1.2.9

  • The printed Tech Cue Sheet now starts each set on its own page, instead of packing the next set into the leftover space below the previous one. A single song's row also no longer splits across a page break (which could leave its FOH/Lighting notes stranded at the top of the next page without the song name).

June 11, 2026

Versions 1.2.7–1.2.8

  • Editing a metaset's master song no longer demotes it. Saving a master (for example to add Front of House or Lighting notes) used to drop it from the builder and take its members' drop zone with it. The metaset now stays intact.
  • The PRO Report opens the native print dialog, with a preview and save-as-PDF, instead of doing nothing or opening a browser window.
  • The printed Tech Cue Sheet uses a larger, more readable font.
  • Tech Cue Sheets now carry the songs inside a medley. Each member of a metaset appears under its master with its own FOH and Lighting notes, editable on the sheet and pre-filled from the song's own notes.
  • The cue sheet's back link landed on a blank page; it now returns to the show's setlist.
  • The PRO Report opens in its own window with a working Print button, instead of a browser tab.
  • Report a Bug now writes to hello@getsetcraft.com.

June 9, 2026

Version 1.2.6

  • Added Privacy Policy and Terms of Service links, reachable from the Help menu and from Backstage (they open the published versions on getsetcraft.com).

June 8, 2026

Versions 1.2.0–1.2.5

  • Licensing cleanup: the app no longer bundles any copyleft components. PDF rasterization moved from PyMuPDF (AGPL) to PDFium, and the HEIC/HEVC libraries (which pulled in GPL-licensed x265) were dropped from the app, since that work only ever runs on the server. The shipped binary is now permissive/LGPL only.
  • Added an Open-Source Licenses page, reachable from the About panel, the Help menu, Backstage, and the website.
  • Help > About Setcraft now opens the standard About panel (it was an AppKit call made off the main thread, so it silently did nothing).
  • Removed the macOS window tab bar. Setcraft is a single-window app, so the default tabbing (the View > Show Tab Bar item and the empty tab strip) only added confusion.
  • Tutorial copy fixes: the overview's closing card now explains how to reach the demo band (via the Artist Switcher) rather than implying you are already in it; the songs "Type" step no longer claims covers print the original artist (they do not); the Backup tutorial highlights the Backup tab.
  • Lyric and chart books now print, not just PDF export. The desktop Print path honors the lyric/chart book checkboxes instead of dropping them.
  • Photo Import no longer shows "Setlist created" and "Review Matches" before you import anything (a desktop-only gate bug).
  • Backstage: the Account & License tab shows license status and the number of activated additional artist profiles; the Artists tab is wider so profile rows with long band names no longer wrap.
  • The Shows list now has a PRO Report link on each show with a setlist.
  • The dashboard rename (pencil) icon follows the theme color.
  • Tutorials: corrected the set cap (four, not unlimited), the default builder view (two sets, not one), and setlist saving (manual Save, not automatic); fixed the song filter list and the print-dialog wording. Added Lyrics, Charts, Shows filtering, and a Save step, a new Backup & Restore tutorial, and a data safety stop on the overview. Removed the redundant Run Setup Wizard entry.
  • Navigation and page labels are now consistently title-cased ("Build Setlist", "Photo Import") so they read correctly on themes whose fonts are not all-caps.
  • Automatic updates via Sparkle: the app checks the signed appcast feed (getsetcraft.com/appcast.xml) and can download, install, and relaunch a new version itself. Manual check from Help, Check for Updates.
  • Also carries the interim lightweight update check, which only runs on builds that do not have Sparkle, so there is never a double prompt.

June 7, 2026

Version 1.1.0

  • Local Backup: automatic on-device disaster-recovery snapshots, taken on launch and before any destructive action, with full rollback from the Backstage Backup tab.
  • Backstage reorganized into Preferences, Integrations, and Account & License.
  • Paid printouts no longer carry the free-version watermark.
  • A real upgrade prompt at the free song limit, replacing the plain error.
  • PRO Report reachable directly from the show page.
  • Accessibility: visible keyboard focus, screen-reader labels on icon buttons.
  • Broad copy and layout polish from the product audit.

Version 1.0.10

  • Final desktop setlist-drag fix (clone and slip behavior in the WKWebView).
  • License reclaim: "deactivate all other computers".
  • Free-tier nudge when over the 15-song limit.

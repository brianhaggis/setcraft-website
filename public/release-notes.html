What's New in Setcraft

July 3, 2026 Current

Versions 1.4.11–1.4.12
  • Typo is now one of the main themes instead of a hidden one. It sits near the top of the theme list, and its wordmark carries a small sun/moon switch that flips the whole look between light and dark.
  • Report a bug without leaving Setcraft. A tab on the right edge opens a short form where you describe the problem and attach a screenshot, and it sends straight to support. Needs an internet connection.
  • The Mystery themes shelf in Backstage now shows how to unlock each secret look, so there is a hint to chase instead of a blank slot.
  • Print buttons added to the Lyric Book and Chart Book in the Song Library, alongside Download.
  • Backstage now keeps you on the tab you were working in after an action (hiding, renaming, or deleting a band) instead of jumping back to the first tab.
  • Fixed a bug where a band whose name contained an apostrophe could not be deleted or renamed.

July 2, 2026

Version 1.4.10
  • Email your setlists, books, and charts. Every place with an AirDrop button now has an email button beside it, which opens a new message in your Mac's mail app with the PDF already attached. Covers setlists, the lyric and chart books, single-song lyrics and charts, the tech cue sheet, and the PRO report.
  • Song Library export buttons are grouped and labeled: a "Lyric Book" box with Download / AirDrop / Email, and a matching "Chart Book" box, so it is clear which book each button acts on.
  • What's New menu item added under Help (above Check for Updates), which opens the release notes for the running version.

July 1, 2026

Versions 1.4.2–1.4.9
  • Typo theme polish: the self-typing wordmark no longer nudges the nav links on each page load, and the current page's nav link now highlights clearly.
  • Wake Up theme: the CRT scanlines are softer and gently flicker.
  • First-launch welcome dialogs no longer overlap: the beta and trial explainers now appear one at a time instead of stacking.
  • Typo theme: the wordmark still types itself in, but it no longer nudges the nav links each time a page loads.
  • Wake Up theme: softened the CRT scanlines and gave them a subtle occasional flicker.
  • Add Multiple. A new button on the Songs page opens a grid where you can type in a batch of songs at once (title, short name, key, capo) or paste them straight from a spreadsheet. The grid grows as you go, and the whole batch imports in one step. Add lyrics, charts, and tech notes later by editing a song.
  • Short names now read in ALL CAPS everywhere they appear on screen, the way a stage label should. The Typo theme (below) and Diva keep their own casing.
  • New theme: Typo. A black-and-white typewriter look, set entirely in monospace and lowercase on warm paper, with a wordmark that types itself in.
  • Wake Up gets a CRT-terminal makeover: the wordmark and headings glitch, over faint scanlines.
  • The Songs selection bar no longer hides behind the trial banner when you scroll.
  • Sharpie mode (Rock Salt) prints cleanly. Capo lines and between-song notes no longer clip: Rock Salt gets taller slots so a title, its capo, and a wrapping note all fit, and the marker font's tall glyphs stop cropping at the top and bottom.
  • On the song library, selecting songs now tints the whole row, using the same color the setlist builder uses to highlight overlapping songs.
  • The sort arrow in the song library is now visible in every theme (it matches the column-header text color instead of the accent, which vanished on some themes like Yacht Rock).
  • In Backstage's Print/Export tab, the divider above Include lyric book now spans the full width, above the chart-book option too.
  • The Add / Edit Song dialog is wider, with the note fields beside the rest so the whole form fits without scrolling. If you close it with unsaved edits, it now asks whether to save, discard, or keep editing.
  • On the song library, clicking anywhere on a row selects it (except its buttons), and the selection bar stays put at the top of the list and follows you as you scroll, so it never jumps the page or leaves your actions out of reach.
  • On printed setlists, metaset subsongs now print at the same size as the between-song notes instead of towering over them.
  • Backstage is redesigned around a floating left sidebar and a wide, centered canvas, so most settings fit without scrolling. Print options move to their own Print/Export tab with a live sample preview, and unlocked secrets get their own tab.
  • Print options are now toggle switches that show what's on at a glance, with a clear "Show key / capo / notes / band name / venue / date / song numbers" set, plus new controls to hide the band name, venue, or date on a sheet. The venue shortens to fit and expands to its full name when there's room.
  • Sharpie mode now defaults to a scrawled "Rock Salt" marker (with Permanent Marker as an option), in natural title case with the key in parentheses.
  • The song library sorts with a Finder-style arrow showing the active column and direction.
  • A new Licenses menu (View / Manage License, Deactivate This Computer, Deactivate All Other Computers) in the menu bar.
  • Print defaults are shared with the print dialog: set them once in Backstage, or save your choices straight from the dialog. Cancelling a print never changes them.
  • The print dialog gains two options: Hide song numbers for a clean, unnumbered sheet, and Sharpie mode, which prints the setlist in a bold handwritten marker font like it was scrawled on the spot. Both show live in the preview.
  • Action buttons now carry their meaning in their color, consistent across every theme: green to save or create, blue to print or export, red to delete, orange to cancel. Upgrade buttons are green and inviting.
  • The print dialog's preview now shows the trial/free watermark exactly as it will print, with a note that a Pro or Road Warrior license removes it.
  • Fixed the bundled fonts not loading in the print preview, so Sharpie mode and the standard print font both render correctly on screen.
  • The free trial now marks printed and exported sheets with a light diagonal stamp, the same as the free version. Every feature stays open during the trial; a clean, gig-ready sheet is what a Pro or Road Warrior license unlocks.
  • Photo import shows the correct trial allotment: five scans a month, not twenty, and the counter now tracks as you use them.
  • The trial banner stays pinned below the top bar as you scroll, and its button reads "Upgrade to keep using all Pro features."
  • On the builder, the Print, cue sheet, and delete actions are right-aligned in their row.

June 30, 2026

Versions 1.4.0–1.4.1
  • The "See the tiers" button on the trial welcome screen now opens the license page as intended.
  • Pro and Road Warrior appear on the Account tab as side-by-side cards, each with a short description of what it includes, so the difference reads at a glance.
  • Clearer trial banner wording. During the trial you already have every feature, so the banner now points to keeping the full app after the trial rather than unlocking it.
  • New installs now get a free 30-day trial of the full app: every feature, no watermark, no card. After 30 days, Setcraft settles into the free version, and your library, shows, and setlists always stay on your Mac.
  • A small banner shows how many trial days remain, with a one-time welcome that explains how the trial works.
  • If a license is close to expiring, Setcraft now gives you a heads-up beforehand, so the full features never switch off unexpectedly. It is dismissable.
  • Photo import is fairer: a scan only counts against your monthly allotment when it actually reads something. A blurry photo that comes back empty no longer costs you.

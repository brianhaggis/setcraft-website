What's New in Setcraft

June 22, 2026 Current

Versions 1.2.27–1.2.28
  • Assorted visual polish and behind-the-scenes refinements.
  • The Lyric Book and Chart Book buttons now have their own clear descriptions.
  • Behind-the-scenes polish.

June 21, 2026

Versions 1.2.25–1.2.26
  • More of your personalization now carries over across app updates and backups.
  • Tuned one theme's background artwork for better readability.
  • Behind-the-scenes polish.
  • Backstage tabs and the Photo Import intro now sit on a solid panel, so they stay readable on every theme.

June 20, 2026

Versions 1.2.21–1.2.24
  • Several themes now have richer background artwork.
  • Behind-the-scenes polish.
  • Backstage Preferences now use the full window width, so the layout has more room to breathe.
  • The downloaded disk image now opens to a tidier install window, with the app and an Applications shortcut laid out for drag-to-install.
  • AirDrop: send a setlist, tech cue sheet, PRO report, lyric or chart book, or a single song's lyrics or chart straight to a nearby Mac, iPhone, or iPad — right from the Print options, without saving a file first.
  • The PRO Report can now be saved as a PDF (or AirDropped), not just printed.

June 19, 2026

Versions 1.2.17–1.2.20
  • Fixed an occasional macOS alert "beep" that could sound when typing while no text field was focused.
  • Your saved look and preferences now persist more reliably across app updates, backups, and a move to a new computer.
  • Assorted visual polish and behind-the-scenes refinements.
  • The small "?" help markers are gone. Tooltips now appear when you hover over a label, and a new switch in Backstage (Preferences) turns help tooltips on or off, so the screens stay uncluttered.
  • Printed setlists and exported PDFs now use one consistent font, regardless of the theme you have on in the app. No more setlists coming out in a pixel or script font just because of your current look.
  • Backstage: the theme picker tiles are more compact, so the list scrolls less.
  • The song editor (and other pop-up dialogs) open scrolled to the top, so the first field is in view instead of halfway down the page.
  • Added a built-in Help Center. Open it from the Help menu (Setcraft Help) and it comes up in its own window with a searchable set of guides: getting started, songs and lyrics, setlists and shows, tech cue sheets and printing, artist profiles, backup, themes, and a troubleshooting page. It works offline, so the answers are there even when the venue Wi-Fi is not.
  • Fixed a case where adding an artist profile showed the upgrade prompt even though you had a paid seat free. The demo band was being counted against your licensed limit; it no longer is, matching how the rest of the app counts.

See the full history at getsetcraft.com/version-history.

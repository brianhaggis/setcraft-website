What's New in Setcraft

June 18, 2026 Current

Versions 1.2.14–1.2.15
  • The demo band's name no longer shows a rename (pencil) button on the dashboard. The demo is a read-only sample, so its name was never editable; the button is gone to match.
  • Account & License: you can now enter an additional-artist-profile add-on key right on the Account & License tab. It used to live only inside the Manage Artists dialog, which is not where most people went looking for it.
  • When you have a paid artist seat you have not used yet, Backstage now points it out: a note on the Account & License tab and a small count badge on the Artists tab, so an activated-but-uncreated profile does not sit there unnoticed. The badge reads clearly on every theme.

June 16, 2026

Version 1.2.13
  • Free version: only your primary artist profile and the demo band stay available. Extra profiles are greyed in the switcher and need an active license. Choose which profile is your primary in Backstage, even on the free version.
  • Deactivating a license now warns you when you have extra profiles, and lets you set your primary first so you keep the right one.
  • Free version: a setlist is capped at 15 songs (your song library is unchanged; building longer setlists is part of the paid version). The demo band is exempt, so you can try full-length setlists there before you buy.
  • Free version: Bandsintown sync shows as locked.
  • The demo band is now a fuller, read-only sample (45 songs, many with lyrics and a sample number chart, plus a couple of saved reminder notes) and a complete sandbox: build multi-set setlists and tech cue sheets with it on the free version, and its exports print without the free-version watermark so you see the full output. Hide it (and bring it back) from Backstage instead of deleting it.
  • Free-version limits no longer show as persistent banners. The app stays out of your way and explains a limit only at the moment you reach one.

June 15, 2026

Version 1.2.12
  • The "Saved defaults" help tooltip in the builder's Between-Song Reminders panel no longer runs off the left edge of the window.
  • Removed a duplicate "View" menu from the menu bar. macOS already provides one with Enter Full Screen, so Reload now lives under the File menu.

June 14, 2026

Versions 1.2.10–1.2.11
  • The Tech Cue Sheet's "Back to Setlist" link now returns to the setlist builder, on the set you were working on, instead of the read-only show view.
  • "Edit Setlist" on the show view now opens that setlist in the builder ready to edit. It used to open an empty builder.
  • In-app email links (the beta feedback button, the Backstage feedback and add-on-code links, and the Licenses page contact) now open your mail client. They did nothing in the desktop app before.
  • In the setlist builder, the Between-Song Reminders panel moved from beneath the sets into the song library column, so saving reminders no longer shrinks the room for the setlist above it. The reminders and metaset panels are now collapsible, and a saved reminder can be added to the current set with a single click as well as by dragging it in.
  • The builder's saved-setlist bar gained a Tech Cue Sheet button, next to Print / PDF.

See the full history at getsetcraft.com/version-history.

What's New in Setcraft

June 25, 2026 Current

Versions 1.2.43–1.2.45
  • Fixed updates not fully taking effect: the app could keep using a cached copy of its styling and scripts from an earlier version, so some fixes silently didn't apply. Each update now always loads its own fresh assets.
  • Reverted the page-load fade from the previous build: it briefly showed the bare page before on-screen elements (like the Build Setlist prompt) appeared. Pages load directly again, without that flash.
  • Moving between tabs now does a soft fade: the page content fades and gently settles into place while the navigation bar and background stay put.

June 24, 2026

Versions 1.2.33–1.2.42
  • Fixed the unlockable nav switches (and the icons beside them) jumping into place a moment after each page loaded. They were being added by a script just after the page first drew; now they're in place from the first frame, so the nav stays put.
  • The themed wordmark, nav, and headings now load up front, so the nav icons and cards no longer shift into place a frame after each page loads — and without the brief full-window background flash the previous build introduced.
  • Restored a soft fade-in of the page content on navigation (background stays put).
  • Each page now waits for its fonts to be ready and then fades in, which removes the last bit of the one-frame "jump" where icons and cards settled into place after load — and gives a consistent soft fade between pages (this fade now shows even with macOS "Reduce Motion" on).
  • Fixed a one-frame jump on page load where the nav icons/switches and the dashboard cards briefly appeared slightly out of place and then snapped into position as fonts finished loading.
  • Page navigation now does a quick, soft fade instead of a hard cut.
  • Fixed the navigation bar briefly flashing in a larger/placeholder font and then snapping to the correct font on every page — fonts now display correctly from the first frame.
  • The beta welcome's "Don't show this again" choice now sticks between launches (it's saved with your profile instead of in browser storage that could reset).
  • Pages now appear instantly when you move between tabs. The brief slide-and-fade that played on every page is gone — it was replaying on each click (and made the longer Songs and Shows lists look like they stuttered in as they loaded).
  • Smoother navigation: the nav bar no longer flickers or shifts position when you move between tabs (its sizing is now applied as the page first draws, instead of a moment after).
  • Your settings and one-time prompts (like "Don't show this again") now reliably persist between launches; previously some could quietly reset on restart.
  • Theme entrance animations now play once per launch instead of replaying every time you return to the dashboard.
  • Cleaner launch: the window now opens dark instead of flashing white first, and animated themes (including Fireflies) start from the dark wash and resolve into the theme, instead of briefly showing the page before the animation runs.
  • Fixed the Build Setlist screen's dark overlay opening at a smaller size and then snapping to fill the window. (The 1.2.34 attempt didn't fully resolve it; this addresses the actual cause — the overlay was briefly tied to the page's load animation.)
  • Fixed the help tooltip on the New Artist Profile dialog (the Bandsintown "?" icon) not appearing on hover.
  • Fixed the Build Setlist screen briefly showing its dark overlay at the wrong size before snapping to fill the window when it opened.
  • Smoother first paint on the dashboard, so the page resolves cleanly instead of appearing and then animating a beat later.
  • Assorted visual polish and behind-the-scenes refinements.
  • Beta testers: a new Beta Tools tab in Backstage, with a "Reset app state" tool that re-runs first-time setup and restores default settings while keeping all of your songs, shows, and setlists.
  • The beta welcome shown on first launch now has a "Don't show this again" option.

June 23, 2026

Version 1.2.32
  • Fixed an error that could pop up when saving or printing a setlist (the setlist was actually saved, but the app wrongly reported it had failed).
  • The Build Setlist tab now opens with a clear prompt to pick an upcoming show or a past setlist, instead of a dimmed builder.
  • The navigation bar keeps a consistent width across themes and, at smaller window sizes, the links now move to a tidy second row instead of wrapping unevenly.
  • Assorted visual polish and behind-the-scenes refinements.

Versions 1.2.27–1.2.31
  • Assorted visual polish and behind-the-scenes refinements.
  • The Lyric Book and Chart Book buttons now have their own clear descriptions.
  • Behind-the-scenes polish.

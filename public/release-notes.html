What's New in Setcraft

June 15, 2026 Current

Version 1.2.12
  • The "Saved defaults" help tooltip in the builder's Between-Song Reminders panel no longer runs off the left edge of the window.
  • Removed a duplicate "View" menu from the menu bar. macOS already provides one with Enter Full Screen, so Reload now lives under the File menu.

June 14, 2026

Versions 1.2.10–1.2.11
  • The Tech Cue Sheet's "Back to Setlist" link now returns to the setlist builder, on the set you were working on, instead of the read-only show view.
  • "Edit Setlist" on the show view now opens that setlist in the builder ready to edit. It used to open an empty builder.
  • In-app email links (the beta feedback button, the Backstage feedback and add-on-code links, and the Licenses page contact) now open your mail client. They did nothing in the desktop app before.
  • In the setlist builder, the Between-Song Reminders panel moved from beneath the sets into the song library column, so saving reminders no longer shrinks the room for the setlist above it. The reminders and metaset panels are now collapsible, and a saved reminder can be added to the current set with a single click as well as by dragging it in.
  • The builder's saved-setlist bar gained a Tech Cue Sheet button, next to Print / PDF.

June 13, 2026

Version 1.2.9
  • The printed Tech Cue Sheet now starts each set on its own page, instead of packing the next set into the leftover space below the previous one. A single song's row also no longer splits across a page break (which could leave its FOH/Lighting notes stranded at the top of the next page without the song name).

June 11, 2026

Versions 1.2.7–1.2.8
  • Editing a metaset's master song no longer demotes it. Saving a master (for example to add Front of House or Lighting notes) used to drop it from the builder and take its members' drop zone with it. The metaset now stays intact.
  • The PRO Report opens the native print dialog, with a preview and save-as-PDF, instead of doing nothing or opening a browser window.
  • The printed Tech Cue Sheet uses a larger, more readable font.
  • Tech Cue Sheets now carry the songs inside a medley. Each member of a metaset appears under its master with its own FOH and Lighting notes, editable on the sheet and pre-filled from the song's own notes.
  • The cue sheet's back link landed on a blank page; it now returns to the show's setlist.
  • The PRO Report opens in its own window with a working Print button, instead of a browser tab.
  • Report a Bug now writes to hello@getsetcraft.com.

See the full history at getsetcraft.com/version-history.

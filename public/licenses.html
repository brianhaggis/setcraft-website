The software Setcraft is built with

Setcraft is built with open-source software, and we're grateful to the people who make it. The components below are included in or used by the app, each under its own license. Where a license requires it, the full text is available from the linked project and on request from hello@getsetcraft.com.

The app contains no copyleft (GPL or AGPL) components. Everything below is under a permissive license (BSD, MIT, Apache, PSF, and similar) or the LGPL. LGPL libraries are linked dynamically so they can be replaced.

Fonts

Setcraft bundles these typefaces under the SIL Open Font License 1.1 or the Apache License 2.0: Inter, Bebas Neue, Barlow Condensed, Orbitron, Playfair Display, Bangers, Lora, Righteous, DM Serif Display, Abril Fatface, Press Start 2P, VT323, Sacramento, Permanent Marker, Share Tech Mono, Metal Mania, and Bravura / Bravura Text.

Hosted services

Setlist photo import sends your photo to Setcraft's server, which uses Anthropic's API to read it (see the Privacy Policy). The server uses pypdfium2 (PDFium) and pillow-heif to handle PDF and HEIC uploads. These run on our servers and are not part of the app.
